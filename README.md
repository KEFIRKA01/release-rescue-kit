# Release Rescue Kit

Набор для диагностики релизов, health-check и rescue-сценариев.

## Ключевые сценарии
- основная цепочка от заявки до результата;
- административный контур и статусы;
- интеграционный слой и уведомления;
- устойчивость и проверка граничных случаев.

## Что показывает проект
- зрелая бизнес-логика, а не просто UI;
- аккуратные контуры данных и событий;
- реалистичный рабочий процесс.
# Набор стабилизации релизов

Набор для стабилизации веб-проектов после релиза: проверка доступности, смоук-тесты, журнал проблем и чеклист релиза.

## Что показывает проект

- подход с расчётом на production;
- автоматическую проверку ключевых сценариев;
- быстрый отчёт о доступности проекта;
- структурирование симптомов и проблем после релиза;
- загрузку списка целей из seed-файла и сводный отчёт по релизу.
- аварийный сценарий действий для нестандартных сбоев;
- оценку риска сервисного окна и нагрузки владельца процесса.

## Для каких задач подходит

- исправление багов и аварийное восстановление;
- аудит веб-проекта после запуска;
- технический долг;
- стабилизация интеграций;
- поддержка действующего сервиса.
